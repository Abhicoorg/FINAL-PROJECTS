Project Overview

This project focuses on applying data science and machine learning techniques to solve real-world business problems. It includes multiple analytical models such as:

Customer Lifetime Value (CLV) Prediction
Movie Success Prediction & Sentiment Analysis
HR Attrition Prediction
Airbnb Pricing Recommendation System

These projects demonstrate how data can be used to improve business decisions, customer insights, and predictive analytics.

 Objectives
Predict customer lifetime value using machine learning
Analyze customer behavior for better segmentation
Identify key factors influencing business outcomes
Build predictive models for decision-making
Visualize insights using dashboards
Tools & Technologies
Programming: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
NLP: NLTK (VADER Sentiment Analysis)
Visualization: Power BI
Other Tools: ydata-profiling
 
 Project Modules

1️Customer Lifetime Value (CLV) Prediction
Built regression models (Linear Regression, Random Forest)
Used RFM features (Recency, Frequency, Monetary)
Helps identify high-value customers

Key Outcome:
Improved customer segmentation and marketing strategies

2️ Movie Success Prediction & Sentiment Analysis
Predicted movie revenue using regression
Performed sentiment analysis on movie overviews
Used VADER for sentiment scoring

Key Outcome:
Identified factors influencing box office success

3️ HR Attrition Prediction
Built Logistic Regression model
Analyzed employee data (salary, satisfaction, experience)
Predicted employee turnover

Key Outcome:
Helps HR teams reduce attrition and improve retention

4️ Airbnb Pricing Recommendation System
Built pricing prediction model
Analyzed location, availability, and room type
Suggested optimal pricing strategies

Key Outcome:
Helps hosts maximize revenue
